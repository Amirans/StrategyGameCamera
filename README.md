# Strategy Camera Movement in Unreal

This is a Strategy Camera Movement template with features such as Camera Zoom and Edge Scrolling.  The template is available in C++ and Blueprint.

> **Features**
> - Camera Zoom
> - Camera Movement
> - Camera Edge Scrolling
